# Zoom Cues Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Let the author declare per-chunk zoom cues in `script.json`; the engine computes a pure, exact Tella `add_zoom` plan that the make-video skill applies via Tella MCP (Option 2).

**Architecture:** A pure planner (`src/zoom.ts`) turns script chunks with a `zoom` field into a `ZoomPlan` (Tella `add_zoom` geometry, in original-recording milliseconds). A thin CLI (`scripts/plan-zooms.ts`) reads `videos/<slug>/script.json`, writes `videos/<slug>/zoom-plan.json`, and prints a table. The skill applies the plan against the Tella project (clear → add → export → swap → re-run make-video). No Tella coupling in the engine; the plan is pure geometry.

**Tech Stack:** Bun + TypeScript, `bun test`. FFmpeg via existing `src/ffprobe.ts`. Tella MCP (skill-side only, not in engine code).

## Global Constraints

- Package manager: **bun** (never npm). Run scripts via `bun run <script>`, tests via `bun test`.
- Zoom type is always **`manualZoom`** in this slice (steady; no `trackingZoom`).
- A zoom spans the **whole chunk** (`sourceStart`…`sourceEnd`). No sub-chunk ranges.
- Defaults: `scale` = **1.25**, `focusPct` = **[50, 50]** (center).
- Scale is clamped to **[1, 4]** (Tella hard max 4); scale > 1.5 emits a "may feel heavy" warning. `focusPct` clamped to **[0, 100]**.
- `estFinalSec` = `min(span / speedFactor(span, voDuration), voDuration)`, rounded to 1 decimal — the estimated seconds of *moving* (non-frozen) footage the zoom covers in the final cut. Present only when the chunk's VO mp3 already exists; the plan never synthesizes.
- The engine never reads Tella IDs; identity (`videoId`/`clipId`) is resolved skill-side and cached in `videos/<slug>/tella.json` (gitignored under `videos/`).
- Single-clip Tella projects only. Never trim/cut in Tella — zoom is an overlay; trimming shifts the timeline and breaks VO/`script.json` alignment.
- Match existing test style (`tests/align.test.ts`): `import { expect, test } from "bun:test"`, small typed factory helpers, `toBeCloseTo` for floats.

```ts
import type { ScriptChunk, VoChunk, ZoomPlan, ZoomPlanEntry } from "./types";
import { clamp, speedFactor } from "./timing";

const round1 = (n: number) => Math.round(n * 10) / 10;

const clampFocus = (v: number, id: string, axis: string, warnings: string[]): number => {
  const c = clamp(v, 0, 100);
  if (c !== v) warnings.push(`${id}: focus ${axis} ${v} clamped to ${c}`);
  return c;
};

// Pure: turn script chunks that carry a `zoom` cue into Tella add_zoom geometry
// (original-recording milliseconds). No I/O, no Tella coupling.
export function planZooms(script: ScriptChunk[], vo?: VoChunk[]): ZoomPlan {
  const warnings: string[] = [];
  const zooms: ZoomPlanEntry[] = [];

  for (const c of script) {
    if (!c.zoom) continue;

    const span = c.sourceEnd - c.sourceStart;
    if (span <= 0) {
      warnings.push(`${c.id}: skipped — sourceEnd <= sourceStart`);
      continue;
    }

    let scale = c.zoom.scale ?? 1.25;
    if (scale < 1) {
      warnings.push(`${c.id}: scale ${scale} clamped to 1`);
      scale = 1;
    } else if (scale > 4) {
      warnings.push(`${c.id}: scale ${scale} clamped to 4 (Tella max)`);
      scale = 4;
    } else if (scale > 1.5) {
      warnings.push(`${c.id}: scale ${scale} may feel heavy (>1.5)`);
    }

    const [rawX, rawY] = c.zoom.focusPct ?? [50, 50];
    const focusPoint = {
      xPct: clampFocus(rawX, c.id, "x", warnings),
      yPct: clampFocus(rawY, c.id, "y", warnings),
    };

    const entry: ZoomPlanEntry = {
      chunkId: c.id,
      type: "manualZoom",
      startTimeMs: Math.round(c.sourceStart * 1000),
      durationMs: Math.round(span * 1000),
      scale,
      focusPoint,
    };

    const v = vo?.find((x) => x.id === c.id);
    if (v && v.duration > 0) {
      const sf = speedFactor(span, v.duration);
      entry.estFinalSec = round1(Math.min(span / sf, v.duration));
    }

    zooms.push(entry);
  }

  return { zooms, warnings };
}
```

```ts
// I/O wrapper: read script.json from <dir>, probe any cached vo/<id>.mp3 for
// durations (best-effort — never synthesizes), and return the plan.
export async function planZoomsForDir(dir: string): Promise<ZoomPlan> {
  const script = (await Bun.file(`${dir}/script.json`).json()) as ScriptChunk[];
  const vo: VoChunk[] = [];
  for (const c of script) {
    const f = `${dir}/vo/${c.id}.mp3`;
    if (await Bun.file(f).exists()) {
      vo.push({ id: c.id, file: f, duration: await ffprobeDuration(f) });
    }
  }
  return planZooms(script, vo.length ? vo : undefined);
}
```
